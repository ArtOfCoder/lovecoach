# 恋爱进化论 - 上线指南

## 一、小程序快速上线步骤

### Step 1：注册微信小程序账号
1. 访问 https://mp.weixin.qq.com/
2. 点击"立即注册" → 选择"小程序"
3. 填写邮箱、设置密码（需要未注册过公众号的邮箱）
4. 完成邮件验证 → 填写主体信息（个人/企业）
5. 登录后进入小程序管理后台，记录你的 **AppID**

> 💡 **个人注册** 免费，但部分类目（如医疗、金融）需要企业主体。
> 恋爱技巧/情感教育类属于"生活服务 - 情感交友/教育培训"，**个人可注册**。

---

### Step 2：安装微信开发者工具
1. 下载：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 安装并打开
3. 扫码登录（用注册小程序的微信扫码）

---

### Step 3：导入本项目
1. 打开微信开发者工具
2. 点击"+"新建项目
3. 选择目录：`lovecoach-miniapp/`
4. 填入你的 **AppID**（如没有可选"测试号"先体验）
5. 点击"确定"导入

---

### Step 5：本地调试
在开发者工具中：
- 点击"编译"查看效果
- 使用"真机调试"在手机上预览效果
- 检查各页面是否正常显示

---

### Step 6：提交审核上线
1. 开发者工具 → 右上角"上传"
2. 填写版本号（如 1.0.0）和备注
3. 登录 https://mp.weixin.qq.com/
4. 进入"版本管理" → 找到刚上传的版本
5. 点击"提交审核"
6. 填写小程序类目：**生活服务 > 情感交友** 或 **教育 > 在线教育**
7. 等待审核（通常 1-3 个工作日）
8. 审核通过后点击"发布"

## 二、升级为 APP 路线图

### 方案一：Uniapp（推荐，一套代码多端）
- 可以将本小程序代码用 Uniapp 重写
- 支持同时发布：微信小程序 + H5 + iOS + Android
- 成本低，学习曲线平缓

### 方案二：React Native / Flutter
- 纯原生体验
- 开发成本较高
- 适合有专业团队时选用

### APP上线流程（国内安卓）
1. 完成APP开发
2. 软件著作权申请（可选但建议）
3. 在各大应用商店提交（华为、小米、OPPO等）
4. 各渠道审核（1-7天）

### APP上线流程（iOS）
1. 注册苹果开发者账号（$99/年）
2. 打包 .ipa 文件
3. 提交 App Store Connect 审核（1-2天）

## 三、商业化变现方案

### 当前适合的变现模式：

| 方案 | 实现难度 | 预期收益 |
|------|----------|----------|
| **课程付费解锁** | ★☆☆ | 中等，1-9.9元/课 |
| **会员订阅** | ★★☆ | 稳定，18-98元/月 |
| **AI顾问次数包** | ★★☆ | 高频，3-10元/次 |
| **情感测评报告** | ★★☆ | 转化率高 |
| **广告变现** | ★☆☆ | 接入微信流量主，按展示计费 |

### 接入微信支付：
1. 小程序后台 → "支付" → 申请接入
2. 需要企业主体才能开通
3. 开通后在代码中使用 `wx.requestPayment()` API

### 接入流量主（广告）：
1. 小程序月活≥1000 可申请
2. 后台 → "流量主" → 申请
3. 在页面中放置 `<ad>` 组件即可

## 五、技术升级建议

### 接入真实 AI（替换现有规则引擎）

```javascript
// 替换 ai-coach.js 中的 getAIResponse 方法
// 调用腾讯混元 API 示例：
async getAIResponse(text) {
  const res = await wx.cloud.callFunction({
    name: 'ai-chat',
    data: { message: text, history: this.data.messages }
  })
  return { answer: res.result.content, suggestions: [] }
}
```

云函数 `ai-chat/index.js`：
```javascript
// 需要在腾讯云开通混元大模型 API
const Hunyuan = require('@tencentcloud/sdk').Hunyuan
exports.main = async (event) => {
  const client = new Hunyuan({ secretId: '...', secretKey: '...' })
  const result = await client.ChatCompletions({
    Model: 'hunyuan-standard',
    Messages: [
      { Role: 'system', Content: '你是一名专业的恋爱情感顾问...' },
      { Role: 'user', Content: event.message }
    ]
  })
  return { content: result.Choices[0].Message.Content }
}
```
