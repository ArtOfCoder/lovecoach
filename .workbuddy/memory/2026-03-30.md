## 修复问题

### 1. 云开发初始化错误
- **问题**：`errCode: -601002` 云开发环境未初始化
- **原因**：`app.js` 中的 `wx.cloud.init()` 使用了占位符环境 ID `'prod-xxxxxx'`
- **解决**：注释掉云开发初始化代码（当前使用网络请求模式）

### 2. AI 顾问页面问题
- **问题 1**：欢迎语不显示，问问题没有回复
- **问题 2**：内容"闪现"
- **原因**：WXML 中使用 `{{item.role === 'ai'}}` 判断，但 JS 中使用的是 `role: 'assistant'`
- **解决**：修改 WXML 为 `{{item.role === 'assistant' || item.role === 'ai'}}`，兼容旧数据

### 3. API 请求错误
- **问题**：`messages[1].role: unknown variant 'ai'`
- **原因**：DeepSeek API 不支持 `role: 'ai'`，只支持 `system`, `user`, `assistant`, `tool`
- **解决**：将所有 `role: 'ai'` 改为 `role: 'assistant'`
