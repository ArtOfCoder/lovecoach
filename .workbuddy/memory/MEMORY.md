# 恋爱进化论小程序 - 项目记忆

## 项目概述
微信小程序（恋爱教练类），个人主体，需使用本地模式（USE_LOCAL_MODE = true）。
H5 版本保留完整 AI 功能。

## 核心配置

### AI 模式（重要）
- **当前：小程序本地模式（H5版本保留AI）**
- 配置文件：`utils/ai.js`
- 原因：个人主体小程序审核不得使用 AI 深度合成

### 主要页面
- 恋爱顾问 `pages/ai-coach/` - 4种模式（顾问/信号分析/角色练习/倾诉）
- 灵魂伴侣 `pages/soulmate/` - 星座配对+暗黑宇宙风UI+本地契合度引擎
- 聊天回复 `pages/chat-reply/` - 气泡样式+空状态优化
- 课程中心 `pages/course/` - 空状态设计+筛选栏美化
- 个人中心 `pages/profile/` - 成就卡片+菜单列表样式
- 情侣档案 `pages/couple/`

### API 配置（H5版本）
- DeepSeek API：`https://api.deepseek.com/v1/chat/completions`
- 生图：Hunyuan Image 模型

### 技术栈
- 微信小程序原生 + 本地模式
- 云开发（可选，云函数可绕过域名白名单）
- 暗色宇宙风主题（#0d0d1e背景）

### 重要文件
- `utils/ai.js` - AI 统一调用层
- `pages/soulmate/soulmate.js` - 星座契合度引擎
- `pages/soulmate/soulmate.wxml/wxss` - 暗黑宇宙风UI
- `审核说明.md` / `AI_CONFIG.md` / `DOMAIN_CONFIG.md`
