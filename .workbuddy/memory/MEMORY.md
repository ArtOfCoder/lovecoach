# 恋爱进化论小程序 - 项目记忆

## 项目概述
这是一个微信小程序项目（恋爱教练类），包含恋爱顾问、情侣档案、聊天回复建议等功能。
**注意：小程序内所有 AI 字样已清除（审核要求），H5 版本保留 AI 功能。**

## 关键配置

### AI 功能配置（重要）
- **当前模式：本地模式（USE_LOCAL_MODE = true）**
- **位置：** `utils/ai.js`
- **原因：** 个人主体小程序审核要求不得使用 AI 深度合成

### 三种 AI 模式

1. **本地模式（当前使用）**
   - 无需网络
   - 基于关键词匹配
   - 文件：`utils/ai.js` 的 `getLocalAIResponse()` 函数
   - 包含知识库：搭讪、表白、暧昧、分手、约会、冷淡处理、吵架和好、礼物惊喜

2. **云函数模式（备用）**
   - 需要开通微信云开发
   - 部署 `cloudfunctions/ai-proxy`
   - 可绕过域名白名单限制
   - 配置：`USE_CLOUD_FUNCTION = true`, `USE_LOCAL_MODE = false`

3. **网络请求模式（标准）**
   - 需要在微信公众平台配置域名白名单
   - 添加：`https://api.deepseek.com`
   - 配置：`USE_CLOUD_FUNCTION = false`, `USE_LOCAL_MODE = false`

### 主要页面

- **AI 顾问：** `pages/ai-coach/ai-coach.js`
  - 4 种模式：恋爱顾问、信号分析、角色练习、倾诉模式
  - 支持本地模式调用

- **情侣档案：** `pages/couple/couple.js`
- **聊天回复：** `pages/chat-reply/chat-reply.js`
- **课程：** `pages/course/course.js`
- **首页：** `pages/index/index.js`
- **引导页：** `pages/onboarding/onboarding.js`（首次使用引导性别选择）
- **灵魂伴侣：** `pages/soulmate/soulmate.js`（玄学+AI生成+微信支付9.9元解锁）

### API 配置
- DeepSeek API：`https://api.deepseek.com/v1/chat/completions`
- 备用：腾讯混元 API
- API Key 示例：`utils/ai.js` 第 20 行

### 技术栈
- 微信小程序原生开发
- 云开发（可选）
- DeepSeek API（可选）

## 重要文件

- `utils/ai.js` - AI 统一调用层（核心配置文件）
- `pages/ai-coach/ai-coach.js` - AI 顾问页面
- `app.json` - 小程序全局配置
- `AI_CONFIG.md` - AI 配置说明文档
- `cloudfunctions/README.md` - 云函数部署文档

## 用户偏好

- 小程序被第三方平台托管
- 服务器域名功能被禁用
- 需要本地模式解决方案

## 注意事项

1. **域名白名单问题**：由于被托管平台禁用，"不校验合法域名"勾选无效
2. **解决方案**：使用本地模式或部署云函数
3. **未来升级**：如需真实 AI，参考 `AI_CONFIG.md` 和 `cloudfunctions/README.md`
