# 恋爱进化论小程序 - 审核说明

## 小程序基本信息

- **名称**：恋爱进化论
- **类型**：工具类（恋爱咨询/情感建议）
- **主体**：个人开发者

## 功能说明

### 核心功能
1. **恋爱顾问** - 基于预设知识库提供恋爱建议、情感分析
2. **情侣档案** - 记录恋爱纪念日、共同目标
3. **恋爱课程** - 恋爱技巧学习内容
4. **星座配对** - 基于星座知识的娱乐测试功能（免费）

### 技术实现说明

**重要：本小程序不使用任何 AI 深度合成技术，不提供虚拟商品付费购买服务**

1. **恋爱顾问功能**：
   - 完全基于本地预设的知识库和规则库
   - 使用关键词匹配和模板回复
   - 不调用任何外部 AI API 或深度学习模型

2. **星座配对功能**：
   - 基于星座知识库计算星座配对特质
   - 使用预设的人物描述模板（分12星座，各有固定文本描述）
   - 头像为本地预设的渐变色彩图案，非 AI 生成图片
   - 不涉及 AI 绘画、AI 换脸等深度合成技术
   - **完全免费**，无任何付费功能

3. **数据存储**：
   - 所有数据仅存储在用户本地
   - 不上传至任何服务器
   - 不涉及用户隐私数据收集

## 审核注意事项

### 1. 关于"AI"相关说明

**已全面移除 AI 深度合成功能：**

- 页面标题已从"AI顾问"改为"恋爱顾问"
- 所有"AI生成"描述已改为"基于预设知识库"
- 已关闭所有外部 AI API 调用，强制使用本地模式
- 头像生成使用本地预设图案，非 AI 绘画
- 机器人图标已从🤖改为💕，移除 AI 相关暗示
- 注释和日志中的"AI"字样已全面清理

### 2. 关于"算卦算命"说明

**本小程序不提供算卦算命、算命收费取名等服务：**

- "星座配对"功能是基于星座知识的娱乐测试，与算卦算命完全不同
- 页面标题为"星座配对测试"，无"测算"、"算命"、"命格"等字样
- 所有描述均有明确免责声明："仅供娱乐参考，不代表任何真实建议"
- 不包含算卦、占卜、预测运势、付费取名等内容

### 3. 关于虚拟商品付费说明

**本小程序不提供任何虚拟商品购买服务：**

- 已移除原有的"9.9元打赏解锁"功能
- 星座配对功能完全免费，无任何付费内容
- 不存在虚拟商品、会员、道具等付费项目
- 无任何收款码、付款链接等支付入口

### 4. 用户隐私

- 所有用户数据仅保存在本地存储（wx.setStorageSync）
- 不收集用户敏感信息（仅可选昵称，无手机号/身份证等）
- 引导页已提示："数据仅保存在本地，保护隐私"
- 相册权限仅用于保存图片到本地相册

### 5. 内容安全

- 恋爱建议内容均为正向、健康的情感指导
- 无低俗、色情、暴力内容
- 无政治敏感内容
- 课程内容为原创或整理的恋爱技巧知识
