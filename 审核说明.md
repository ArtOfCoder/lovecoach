# 恋爱进化论小程序 - 审核说明

## 小程序基本信息

- **名称**：恋爱进化论
- **类型**：工具类（恋爱咨询/情感建议）
- **主体**：个人开发者

## 功能说明

### 核心功能
1. **恋爱顾问** - 基于预设知识库提供恋爱建议、情感分析
2. **情侣档案** - 记录恋爱纪念日、共同目标
3. **恋爱课程** - 恋爱技巧学习内容
4. **灵魂伴侣测算** - 基于星盘算法的娱乐测试功能

### 技术实现说明

**重要：本小程序不使用任何 AI 深度合成技术**

1. **恋爱顾问功能**：
   - 完全基于本地预设的知识库和规则库
   - 使用关键词匹配和模板回复
   - 不调用任何外部 AI API 或深度学习模型

2. **灵魂伴侣功能**：
   - 基于本地星盘算法计算星座匹配
   - 使用预设的人物描述模板
   - 头像为本地生成的渐变色彩图案，非 AI 生成图片
   - 不涉及 AI 绘画、AI 换脸等深度合成技术

3. **数据存储**：
   - 所有数据仅存储在用户本地
   - 不上传至任何服务器
   - 不涉及用户隐私数据收集

## 审核注意事项

### 1. 关于"AI"相关说明

**已全面移除 AI 深度合成功能：**

- 页面标题已从"AI顾问"改为"恋爱顾问"
- 所有"AI生成"描述已改为"基于预设知识库"
- 已关闭所有外部 AI API 调用，强制使用本地模式
- 头像生成使用本地算法，非 AI 绘画
- 机器人图标已从🤖改为💕，移除AI相关暗示
- 所有"AI自动"描述已改为"自动"或"系统"

### 2. 打赏功能说明

**灵魂伴侣功能采用"打赏解锁"模式：**

- 该功能为**娱乐性质**的星盘测试，非商品销售
- 用户"打赏"后可查看本地算法生成的测试结果
- 使用个人收款码方式，属于用户自愿打赏行为
- 已添加明确的免责声明："本功能基于本地星盘算法生成，仅供娱乐参考"

**合规性说明：**
- 未使用微信支付商户接口（个人开发者无法申请）
- 未进行经营性收款，属于自愿打赏模式
- 已明确标注"仅供娱乐参考"

### 3. 用户隐私

- 所有用户数据仅保存在本地存储（wx.setStorageSync）
- 不收集用户敏感信息（仅可选昵称，无手机号/身份证等）
- 引导页已提示："数据仅保存在本地，保护隐私"
- 相册权限仅用于保存图片到本地相册

### 4. 内容安全

- 恋爱建议内容均为正向、健康的情感指导
- 无低俗、色情、暴力内容
- 无政治敏感内容
- 课程内容为原创或整理的恋爱技巧知识
