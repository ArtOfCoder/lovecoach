## H5版本部署步骤

### 方式1：Vercel部署（推荐，免费）

1. **准备代码**
   ```bash
   # 将 h5-version 目录推送到GitHub
   cd h5-version
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/yourusername/lovecoach-h5.git
   git push -u origin main
   ```

2. **Vercel部署**
   - 访问 https://vercel.com
   - 使用GitHub账号登录
   - 点击 "New Project"
   - 导入 `lovecoach-h5` 仓库
   - Framework Preset 选择 "Other"
   - 点击 "Deploy"

3. **获得访问链接**
   - 部署完成后，Vercel会提供一个 `.vercel.app` 域名
   - 示例：`https://lovecoach-h5.vercel.app`

### 方式2：腾讯云COS部署（国内访问快）

1. **创建存储桶**
   - 登录 [腾讯云控制台](https://console.cloud.tencent.com/cos)
   - 创建存储桶，选择"公有读私有写"
   - 地域选择离你最近的

2. **开启静态网站**
   - 进入存储桶 → 基础配置 → 静态网站
   - 开启静态网站功能
   - 索引文档设置为 `index.html`
   - 错误文档设置为 `index.html`

3. **上传文件**
   - 进入存储桶 → 文件列表
   - 上传 `h5-version` 目录下的所有文件
   - 保持目录结构

4. **访问链接**
   - 静态网站URL格式：`https://<bucket-name>.cos-website.<region>.myqcloud.com`

### 方式3：GitHub Pages部署（免费）

1. **创建GitHub仓库**
   - 将 `h5-version` 目录内容推送到GitHub仓库

2. **开启GitHub Pages**
   - 进入仓库 → Settings → Pages
   - Source 选择 "Deploy from a branch"
   - Branch 选择 "main" / "root"
   - 点击 Save

3. **访问链接**
   - 等待几分钟后，访问 `https://yourusername.github.io/repo-name`

## 小程序版本提交审核

### 已完成的修改

1. ✅ 关闭所有外部AI API调用（`USE_LOCAL_MODE = true`）
2. ✅ 修改页面标题："AI顾问" → "恋爱顾问"
3. ✅ 移除所有"AI生成"字样
4. ✅ 机器人图标从🤖改为💕
5. ✅ 更新免责声明
6. ✅ 修改"AI自动"为"自动"或"系统"

### 提交审核步骤

1. **微信开发者工具**
   - 打开项目
   - 点击 "上传"
   - 填写版本号（如：1.0.1）
   - 填写项目备注

2. **微信公众平台**
   - 登录 [微信公众平台](https://mp.weixin.qq.com)
   - 进入 "版本管理"
   - 找到刚上传的版本
   - 点击 "提交审核"

3. **填写审核信息**
   - 功能页面：选择主要页面（首页、恋爱顾问、灵魂伴侣）
   - 审核说明：参考 `审核说明.md` 文件内容

4. **等待审核**
   - 通常1-3个工作日
   - 审核结果会发送到管理员微信
